# The **same syntax**

````md magic-move
```python314
name = "World"
greeting = f"Hello, {name}!"
```
```python314
name = "World"
greeting = t"Hello, {name}!"
```
````

# Really, the **same syntax**

````md magic-move
```python314
price = 14.95

```
```python314
price = 14.95

```
````

````md magic-move
```
````
